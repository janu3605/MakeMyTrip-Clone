# MakeMyTrip Clone

Welcome to the MakeMyTrip clone project! This application allows users to browse and book flights and hotels with a modern, responsive UI. It also includes an admin dashboard for full platform management.

## Key Features

- **Live Flight Status:** Get real-time updates and track the status of flights instantly.
- **Dynamic Pricing Engine:** Intelligent pricing model that adjusts based on seasonal rules, demand, and admin configurations.
- **Cancellation & Refund System:** Seamless handling of user cancellations with automated refund processing logic.
- **Interactive Seat & Room Selection:** Visual, interactive maps and grids for users to pick their exact seats on flights or preferred rooms in hotels.
- **Comprehensive Review & Rating System:** Users can leave detailed feedback, and admins can effectively moderate flagged reviews.
- **Personalized Recommendations:** Smartly recommends relevant flights, hotels, or alternative choices tailored to user preferences.

## Workflow for Evaluators
1. Start by browsing flights or hotels on the main page to see the **Personalized Recommendations** in action.
2. Proceed to a detail page to experience the **Interactive Seat & Room Selection**. Check real-time data or the **Live Flight Status**.
3. Select your preferred options and proceed with the booking. You can test the **Cancellation & Refund System** by cancelling a booking later from your profile.
4. Leave a review to test the **Comprehensive Review & Rating System**.
5. Sign in using test credentials (or sign up for a new account) to complete bookings.
6. Go to the **Admin Tab** (`/admin`) to evaluate administrative capabilities. From here, you can add new flights or hotels, edit existing entries, configure rules for the **Dynamic Pricing Engine**, and manage flagged user reviews.
